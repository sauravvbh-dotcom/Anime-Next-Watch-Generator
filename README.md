# Anime Next Watch Generator 🧧

**Anime Next Watch** is a web application designed for fans who struggle to decide what to watch next. By leveraging real-time data from MyAnimeList, the app provides a curated experience to discover trending shows, filter by specific genres, and explore top-rated classics.

## 🚀 Features
* **Trending Now:** A dynamic list of the most popular anime currently airing.
* **Smart Filtering:** Filter recommendations by genre (Action, Sci-Fi, Slice of Life, etc.) or "Top Rated" status.
* **Synopsis Cards:** Interactive expandable cards providing a deep dive into the plot, studio, and score without leaving the page.
* **Search Functionality:** Quickly find specific titles using the Jikan search endpoint.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), and Modern JavaScript (ES6+).
* **API:** [Jikan API](https://docs.api.jikan.moe/) (Unofficial MyAnimeList API).
* **Version Control:** Git & GitHub.

## 📂 Project Structure
* `index.html` - The main entry point and UI structure.
* `style.css` - Custom styling and responsive design.
* `script.js` - Logic for fetching API data, DOM manipulation, and filtering.
